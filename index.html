Resources

Advertisements
Ads are the primary representation of what the end-user sees. They are normal Image-Text but can be of other types too

Resource Description
GET https://api.adcloud.com/v2/advertisements List advertisements
GET https://api.adcloud.com/v2/advertisements/:id Show a single advertisement
POST https://api.adcloud.com/v2/advertisements Create a new advertisement
PUT https://api.adcloud.com/v2/advertisements/:id Update an advertisement
DELETE https://api.adcloud.com/v2/advertisements/:id Destroy an advertisement

Campaigns
A campaign is a collection of ad-media and settings that define how a product will be displayed to users on subscribing websites.

Resource Description
GET https://api.adcloud.com/v2/campaigns List campaigns
GET https://api.adcloud.com/v2/campaigns/:id Show a campaign
POST https://api.adcloud.com/v2/campaigns Create a campaign
PUT https://api.adcloud.com/v2/campaigns/:id Update a campaign
PUT https://api.adcloud.com/v2/campaigns/:id/request_approval Set the Campaign Status to Request Approval
PUT https://api.adcloud.com/v2/campaigns/:id/end Set the Campaign Status to Ended
PUT https://api.adcloud.com/v2/campaigns/:id/offer Set the Campaign Status to Offer

Oauth Requests
In order to gain access to the adcloud public api you must have request an access_token via 2 step Oauth.

Resource Description
POST https://api.adcloud.com/v1/oauth/access_token
