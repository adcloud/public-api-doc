POST https://api.adcloud.com/v2/advertisements
Create a new advertisement

Create a new Advertisement

Supported Formats

json

Examples

Request: 
POST https://api.adcloud.com/v2/advertisements/?access_token={ACCESS_TOKEN}
Response:
json
{
   "id":2,
   "product_id":2,
   "type":1,
   "advertisement_design_id":1,
   "target_url":"http://example.org",
   "name":"AdCloud Basic Ad",
   "logo":false,
   "image_text":true,
   "flash":false,
   "display":true,
   "tag":true,
   "postview_type":1,
   "tag_html":"haml",
   "image_alt":"bar",
   "text_headline":"Super Ads",
   "text_body":"AdCloud is an awesome company",
   "text_link":"linklink",
   "postview_url":"http://example.org",
   "keywords":null,
   "exclusion_keywords":null,
   "machine_keywords":null,
   "locations":null,
   "modified":"2012-08-29T13:26:06Z",
   "created":"2012-08-29T13:26:06Z",
   "_meta":{
      "type":"Advertisement",
      "message":"created",
      "status":201,
      "uuid":"93cb6d2fea6538894eac6ffc6e6c85ee"
   }
}

Params

Param name Description
product_id
*required

The Id of the product that is associated with this Advertisement

value: Must be Integer
name
*required

The name by which this Advertisement will be known

value: Must be String
type
*required

The Type of Advertisement to create: Ad:0, Click Tag:1

value: Must be String
one of: 0,1
image_alt
optional

The "Alt" text to use for the Image (if present) for this Advertisement

value: Must be String
text_headline
optional

The Headline that will be displayed for this Advertisement

value: Must be String
text_body
optional

The primary body text that will be used for this Advertisement

value: Must be String
text_link
optional

The Text that will be displayed in the postview_url link on the Advertisement

value: Must be String
tag_html
optional

Custom HTML/Script for this Advertisement

value: Must be String
postview_url
*required

The postview URL can be enhanced by the variable [TIMESTAMP]

value: Must be String
advertisement_design_id
*required

The Id of the Ad Design to be used for this Advertisement

value: Must be Integer
logo
*required

Does this Advertisement make use of a Company Logo

value: Must be Integer
one of: 0,1
image_text
*required

Should be 1 if the ad contains image and text data and 0 if not

value: Must be Integer
one of: 0,1
flash
*required

Does this Advertisement use AdobeFlash instead of an Image?

value: Must be Integer
one of: 0,1
display
*required

Should be 1 if this Advertisement is just an Image. I.e. no text and not flash

value: Must be Integer
one of: 0,1
tag
optional

If this Advertisement contains a custom script or html tag, this value should be "1". An example would be an ad wich is hosted by google adwords.

value: Must be Integer
one of: 0,1
postview_type
*required

Define if a custom tracking pixel should be loaded and of which type 0:none, 1:image, 2:iframe

value: Must be Integer
one of: 0,1,2
keywords
optional

A list of keywords to associate this Advertisement with

value: Must be String
exclusion_keywords
optional

A list of keywords used to ensure that this Advertisement is not associated with

value: Must be String
machine_keywords
optional

Keywords used for mobile advertising

value: Must be String
locations
optional

A set of Geographic Location names to display this Advertisement in

value: Must be String
>>>
