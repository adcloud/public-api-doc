PUT https://api.adcloud.com/v2/campaigns/:id/offer
Set the Campaign Status to Offer

If you wanted to initiate an Offer for a Campaign you would set the Campaign status to "Offer":#{Sisyphus::CampaignStatus::STATES[:offer]}.

Examples

Request: 
PUT https://api.adcloud.com/v2/campaigns/?access_token={ACCESS_TOKEN}
Response:
json
{
   "_meta":{
       "message":"status updated",
       "status":200,
       "uuid":"93cb6d2fea6538894eac6ffc6e6c85ee"
   }
}

Supported Formats

json

Params

Param name Description
id
*required

The Id of the Campaign to be shown

value: Must be Integer
