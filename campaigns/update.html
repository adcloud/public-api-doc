PUT https://api.adcloud.com/v2/campaigns/:id
Update a campaign

If you wish to update a number of a Campaigns settings you would use this method

Examples

Request: 
PUT https://api.adcloud.com/v2/campaigns/?access_token={ACCESS_TOKEN}
Response:
json
{
   "_meta":{
       "message":"updated",
       "status":200,
       "uuid":"93cb6d2fea6538894eac6ffc6e6c85ee"
   }
}

Supported Formats

json

Params

Param name Description
id
*required

The Id of the Campaign to be shown

value: Must be Integer
